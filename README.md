Результат доступен по адресу http://localhost:8080.

## ⚙️ Как запустить проект
Клонировать репозиторий:
```
git clone https://github.com/timarkh01/nginx-lab-4.git
cd nginx-lab-4
```
Запустить контейнеры:

```
docker-compose up -d --build
```
Открыть в браузере: ```http://localhost:8080``` 📂 Содержимое проекта

```docker-compose.yml``` — описание сервиса Nginx

```www/form.html``` — главная HTML-страница

```nginx/default.conf``` — файл для обработки PHP

✅ Результат Сервер в Docker успешно запущен, данные отображаются из публичного API
